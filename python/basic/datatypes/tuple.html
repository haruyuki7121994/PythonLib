Introduction

Trong Python, tuple tương tự như list Sự khác biệt là với tuple chúng ta không thể thêm, xóa, sửa các phần tử sau khi nó được gán trong khi list có thể tự do thực hiện trên đó.

Các đặc điểm của tuple

  • Các phần tử không thể thay đổi
  • Danh sách được sắp xếp theo thứ tự ,có số lượng xác định, phân tách nhau bằng dấu ,
  • Các phần tử được lập index và index đầu tiên là 0
  • Các phần tử có thể là nhiều kiểu dữ liệu khác nhau như int string object etc...

Create Tuple

Các Tuple được tạo bằng cách đặt những giá trị phân tách bằng dấu , Có hoặc không sử dụng dấu () để nhóm dữ liệu.

Note: Việc tạo tuple mà không sử dụng () được gọi là Tuple Packing có số lượng ko hạn định và thuộc bất kỳ kiểu dữ liệu nào.

# Different types of tuples

# Empty tuple
# Output: ()
my_tuple = ()
print(my_tuple)

# Tuple having integers
# Output: (1, 2, 3)
my_tuple = (1, 2, 3)
print(my_tuple)

# tuple with mixed datatypes
# Output: (1, 'Hello', 3.4)
my_tuple = (1, "Hello", 3.4)
print(my_tuple)

# nested tuple
# Output: ('mouse', [8, 4, 6], (1, 2, 3))
my_tuple = ("mouse", [8, 4, 6], (1, 2, 3))
print(my_tuple)
# Output: (3, 4.6, 'dog')
my_tuple = 3, 4.6, "dog"
print(my_tuple)

# tuple unpacking is also possible
a, b, c = my_tuple

print(a)      # 3
print(b)      # 4.6
print(c)      # dog

Tạo tuple với một phần tử có một chút khó khăn. Chúng ta sẽ cần một dấu , ở cuối để chỉ định cho Python hiểu rằng đây là một tuple

my_tuple = ("hello")
print(type(my_tuple))  # <class 'str'>

# Creating a tuple having one element
my_tuple = ("hello",)
print(type(my_tuple))  # <class 'tuple'>

# Parentheses is optional
my_tuple = "hello",
print(type(my_tuple))  # <class 'tuple'>

Access Tuple

Một vài cách để truy cập các phần tử trong tuple

  • Indexing
  • Slicing
  • Unpacking

1. Indexing

  • Chỉ mục phải là số integer có thể âm hoặc dương. Không thể sử dụng kiểu float hoặc các kiểu khác nếu không sẽ báo lỗi TypeError
  • Nếu truy cập index ngoài phạm vi sẽ báo lỗi IndexError
  • Danh sách lồng truy cập bằng cách sử dụng index lồng
# Accessing tuple elements using indexing
my_tuple = ('p','e','r','m','i','t')

print(my_tuple[0])   # 'p' 
print(my_tuple[5])   # 't'

# IndexError: list index out of range
# print(my_tuple[6])

# Index must be an integer
# TypeError: list indices must be integers, not float
# my_tuple[2.0]

# nested tuple
n_tuple = ("mouse", [8, 4, 6], (1, 2, 3))

# nested index
print(n_tuple[0][3])       # 's'
print(n_tuple[1][1])       # 4
# Negative indexing for accessing tuple elements
my_tuple = ('p', 'e', 'r', 'm', 'i', 't')

# Output: 't'
print(my_tuple[-1])

# Output: 'p'
print(my_tuple[-6])

2. Slicing

Chúng ta có thể truy cập nhiều phần tử trong tuple bằng cách sử dụng toán tử :

# Accessing tuple elements using slicing
my_tuple = ('p','r','o','g','r','a','m','i','z')

# elements 2nd to 4th
# Output: ('r', 'o', 'g')
print(my_tuple[1:4])

# elements beginning to 2nd
# Output: ('p', 'r')
print(my_tuple[:-7])

# elements 8th to end
# Output: ('i', 'z')
print(my_tuple[7:])

# elements beginning to end
# Output: ('p', 'r', 'o', 'g', 'r', 'a', 'm', 'i', 'z')
print(my_tuple[:])

3. Unpacking

# Tuple Unpacking
tuple1 = ('good', 'morning', 'everyone')
a, b, c = tuple1

# Output: 'good'
print(a)

# Output: 'morning'
print(b)

# Output: 'everyone'
print(c)

Change Tuple

Tuple là bất biến. Có nghĩa rằng các phần tử trong tuple không thể thay đổi ngay khi được chỉ định. Nhưng, nếu phần tử của nó là kiểu có thể thay đổi như list thì có thể thay đổi.

Chúng ta có thể chỉ định lại bằng những một tuple khác.

# Changing tuple values
my_tuple = (4, 2, 3, [6, 5])

# TypeError: 'tuple' object does not support item assignment
# my_tuple[1] = 9

# However, item of mutable element can be changed
my_tuple[3][0] = 9    # Output: (4, 2, 3, [9, 5])
print(my_tuple)

# Tuples can be reassigned
my_tuple = ('p', 'r', 'o', 'g', 'r', 'a', 'm', 'i', 'z')

# Output: ('p', 'r', 'o', 'g', 'r', 'a', 'm', 'i', 'z')
print(my_tuple)

Sử dụng toán tử + để nối 2 tuple. Sử dụng toán tử * để lập lại số lần của 1 tuple.

# Concatenation
# Output: (1, 2, 3, 4, 5, 6)
print((1, 2, 3) + (4, 5, 6))

# Repeat
# Output: ('Repeat', 'Repeat', 'Repeat')
print(("Repeat",) * 3)

Delete Tuple

Như chúng ta đã biết tuple là bất biến nên không thể xóa một phần tử. Chỉ có thể xóa toàn bộ tuple.

# Deleting tuples
my_tuple = ('p', 'r', 'o', 'g', 'r', 'a', 'm', 'i', 'z')

# can't delete items
# TypeError: 'tuple' object doesn't support item deletion
# del my_tuple[3]

# Can delete an entire tuple
del my_tuple

# NameError: name 'my_tuple' is not defined
print(my_tuple)

Tuple Methods

Chỉ có 2 phương thức trong tuple đó là index() và count()

  • count(): đếm số lần xuất hiện của 1 phần tử
  • index(): nhận vào giá trị của và trả về index
my_tuple = ('a', 'p', 'p', 'l', 'e',)

print(my_tuple.count('p'))  # Output: 2
print(my_tuple.index('l'))  # Output: 3

Tuple Operations

# Membership test in tuple
my_tuple = ('a', 'p', 'p', 'l', 'e',)

# In operation
print('a' in my_tuple)
print('b' in my_tuple)

# Not in operation
print('g' not in my_tuple)
# Using a for loop to iterate through a tuple
for name in ('John', 'Kate'):
    print("Hello", name)

#Output: Hello John
         Hello Kate

Ưu điểm của tuple so với list

Vì tuple khá giống với list nên cả 2 đều được sử dụng trong những tình huống tương tự nhau. Tuy nhiên, có một số ưu điểm của sử dụng tuple so với list được liệt kê dưới đây:

  • Thường sử dụng tuple có các kiểu dữ liệu không đồng nhất. Sử dụng list cho các kiểu dữ liệu đồng nhất.
  • Tuple là bất biến nên việc loop qua các giá trị sẽ nhanh hơn với list. Vì vậy hiệu suất cũng tăng lên.
  • Các phần tử trong Tuple là bất biến nên có thể được sử dụng làm key cho dictionary Với list điều này là không thể.
  • Nếu bạn có dữ liệu không thay đổi, việc riển khai tuple sẽ đảm bảo rằng nó sẽ được bảo vệ chống sửa.
