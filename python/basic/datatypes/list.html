Introduction

List cũng giống như mảng có kích thước động. Các phần tử trong List không cần phải đồng nhất, điều này làm cho nó trở thành một công cụ mạnh mẽ và sử dụng nhiều trong Python.

Các đặc điểm của List

  • Kích thước có thể thay đổi
  • Danh sách được sắp xếp theo thứ tự và có số lượng xác định
  • Các phần tử được lập index và index đầu tiên là 0
  • Các phần tử có thể là nhiều kiểu dữ liệu khác nhau như int string object etc...
  • Các phần tử có thể được thay đổi ngay cả sau khi tạo, khác với String Tuple Set

Create List

List có thể được tạo bằng cách đặt nhiều kiểu dữ liệu giống hoặc khác nhau bên trong dấu ngoặc vuông [], phân cách nhau bằng ,

List không cần một hàm built-in để khởi tạo, các phần tử bên trong có thể thay đổi.

# empty list
my_list = []

# list of integers
my_list = [1, 2, 3]

# list with mixed data types
my_list = [1, "Hello", 3.4]

# nested list
my_list = ["mouse", [8, 4, 6], ['a']]

Access List

Sử dụng index để truy cập một phần tử trong danh sách

Các đặc điểm của index trong List:

  • Chỉ mục phải là số integer có thể âm hoặc dương. Không thể sử dụng kiểu float hoặc các kiểu khác nếu không sẽ báo lỗi TypeError
  • Nếu truy cập index ngoài phạm vi sẽ báo lỗi IndexError
  • Danh sách lồng truy cập bằng cách sử dụng index lồng
# List indexing

my_list = ['p', 'r', 'o', 'b', 'e']

# Output: p
print(my_list[0])

# Output: o
print(my_list[2])

# Output: e
print(my_list[4])

# Nested List
n_list = ["Happy", [2, 0, 1, 5]]

# Nested indexing
print(n_list[0][1])

print(n_list[1][3])

# Error! Only integer can be used for indexing
print(my_list[4.0])
# Negative indexing in lists
my_list = ['p','r','o','b','e']

# Output: e
print(my_list[-1])

# Output: p
print(my_list[-5])

Add List

Một số phương pháp sử dụng để thêm 1 hoặc nhiều phần tử vào danh sách:

  • append(): thêm một phần tử vào cuối danh sách
  • insert(): thêm một phần tử vào vị trí cụ thể
  • extend(): thêm nhiều phần tử vào cuối danh sách
  • Sử dụng toán tử + để hợp 2 danh sách. Gọi là concatenation
  • Sử dụng toán tử * để lập lại số lần của danh sách
  • Sử dụng Slice để thêm 1 hoặc nhiều phần tử tại vị trí cụ thể
# Appending and Extending lists in Python
odd = [1, 3, 5]

odd.append(7)

print(odd)
>>> [1, 3, 5, 7]

odd.extend([9, 11, 13])

print(odd)
>>> [1, 3, 5, 7, 9, 11, 13]
# Concatenating and repeating lists
odd = [1, 3, 5]

print(odd + [9, 7, 5])
>>> [1, 3, 5, 9, 7, 5]

print(["re"] * 3)
>>> ['re', 're', 're']
# Demonstration of list insert() method
odd = [1, 9]
odd.insert(1,3)

print(odd)
>>> [1, 3, 9]

odd[2:2] = [5, 7]

print(odd)
>>> [1, 3, 5, 7, 9]

Change List

List có thể thay đổi bằng cách sử dụng toán tử = để thay đổi giá trị của 1 hoặc nhiều phần tử

# Correcting mistake values in a list
odd = [2, 4, 6, 8]

# change the 1st item    
odd[0] = 1

print(odd)
>>> [1, 4, 6, 8]

# change 2nd to 4th items
odd[1:4] = [3, 5, 7]

print(odd)
>>> [1, 3, 5, 7]

Delete/Remove List

Một số phương pháp sử dụng để xóa 1 hoặc nhiều phần tử hoặc toàn bộ danh sách:

  • del: xóa 1, nhiều hoặc toàn bộ phần tử
  • remove(): xóa một phần tử, nhận giá trị làm tham số input
  • pop(): xóa một phần tử và trả về giá trị cuối cùng nếu index không được cung cấp. Điều này giúp triển khai List dưới dạng Stack
  • Sử dụng Slice để xóa 1 hoặc nhiều phần tử bằng cách gán một List trống
# Deleting list items
my_list = ['p', 'r', 'o', 'b', 'l', 'e', 'm']

# delete one item
del my_list[2]
print(my_list)
>>> ['p', 'r', 'b', 'l', 'e', 'm']

# delete multiple items
del my_list[1:5]
print(my_list)
>>> ['p', 'm']

# delete entire list
del my_list

# Error: List not defined
print(my_list)
>>> Traceback (most recent call last):
File "<string>", line 18, in <module>
NameError: name 'my_list' is not defined
my_list = ['p','r','o','b','l','e','m']
my_list.remove('p')

# Output: ['r', 'o', 'b', 'l', 'e', 'm']
print(my_list)

# Output: 'o'
print(my_list.pop(1))

# Output: ['r', 'b', 'l', 'e', 'm']
print(my_list)

# Output: 'm'
print(my_list.pop())

# Output: ['r', 'b', 'l', 'e']
print(my_list)

my_list.clear()

# Output: []
print(my_list)
>>> my_list = ['p','r','o','b','l','e','m']
>>> my_list[2:3] = []
>>> my_list
['p', 'r', 'b', 'l', 'e', 'm']
>>> my_list[2:5] = []
>>> my_list
['p', 'r', 'm']

Slice List

Syntax: [start : stop : steps]
Parameters: 
     start: index bắt đầu, default là 0
     end: index kết thúc, default là index cuối cùng của danh sách
     step: bước nhảy lập qua danh sách, default là 1
# List slicing in Python
my_list = ['p','r','o','g','r','a','m','i','z']

# elements 3rd to 5th
print(my_list[2:5])
>>> ['o', 'g', 'r']

# elements beginning to 4th
print(my_list[:-5])
>>> ['p', 'r', 'o', 'g']

# elements 6th to end
print(my_list[5:])
>>> ['a', 'm', 'i', 'z']

# elements beginning to end
print(my_list[:])
>>> ['p', 'r', 'o', 'g', 'r', 'a', 'm', 'i', 'z']

#  below list has numbers from 2 to 8 in step 2
print(my_list[1:8:2])
>>> ['r', 'g', 'a', 'i']

List Methods

Các methods có sẵn của List được liệt kê theo bảng dưới đây:

Python List Methods
append() - Thêm phần tử vào cuối danh sách
extend() - Thêm tất cả phần tử từ một list vào một list khác
insert() - Thêm phần tử với index cụ thể
remove() - Xóa phần tử trong danh sách
pop() - Xóa và trả về phần tử với index cụ thể
clear() - Xóa tất cả phần tử trong danh sách
index() - Trả về index đầu tiên khớp với phần tử
count() - Đếm số phần tử xuất hiện trong danh sách
sort() - Sắp xếp danh sách theo thứ tự tăng dần
reverse() - Đảo ngược thứ tự trong danh sách
copy() - Copy danh sách

List Operations

  • Sử dụng từ khóa in để kiểm tra một phần tử có tồn tại trong List hay không
  • Sử dụng for để lập qua từng phần tử trong List
my_list = ['p', 'r', 'o', 'b', 'l', 'e', 'm']

# Output: True
print('p' in my_list)

# Output: False
print('a' in my_list)

# Output: True
print('c' not in my_list)
for fruit in ['apple','banana','mango']:
print("I like",fruit)
>>> I like apple
I like banana
I like mango
