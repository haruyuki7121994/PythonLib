Introduction

Set là một tập hợp các phần tử không có thứ tự, các phần tử là duy nhất (không trùng lặp) và các phần tử phải là bất biến (không thể thay đổi)

Các đặc điểm của set

  • Bản thân set là một tập hợp có thể thay đổi. Chúng ta có thể thêm hoặc xóa các phần tử. Nhưng không thể sửa một phần tử vì là bất biến
  • Set có các phép toán về tập hợp riêng như union intersection symmetric difference etc...
  • Các phần tử có thể là nhiều kiểu dữ liệu khác nhau như int string tuple etc... Nhưng không thể là các phần tử có thể thay đổi như list set dictionary

Create Set

Có 2 cách để tạo một set:

  • Sử dụng hàm built-in set() với một đối tượng có thể lặp lại
  • Đặt những phần tử bên trong dấu {} phân tách nhau bằng dấu ,

Lưu ý: Một set không thể chứa các phần tử có thể thay đổi như list set dictionary

# Different types of sets in Python
# set of integers
# Output: {1, 2, 3}
my_set = {1, 2, 3}
print(my_set)

# set of mixed datatypes
# Output: {1.0, (1, 2, 3), 'Hello'}
my_set = {1.0, "Hello", (1, 2, 3)}
print(my_set)
# set cannot have duplicates
# Output: {1, 2, 3, 4}
my_set = {1, 2, 3, 4, 3, 2}
print(my_set)

# we can make set from a list
# Output: {1, 2, 3}
my_set = set([1, 2, 3, 2])
print(my_set)

# set cannot have mutable items
# here [3, 4] is a mutable list
# this will cause an error.

my_set = {1, 2, [3, 4]}

Tạo một set trống có một chút khó khăn với dấu {} vì sẽ tạo ra một dictionary trống. Thay vào đó để tạo một empty set chúng ta sử dụng hàm set() ko có đối số bên trong.

# Distinguish set and dictionary while creating empty set

# initialize a with {}
a = {}

# check data type of a
# Output: <class 'dict'>
print(type(a))

# initialize a with set()
a = set()

# check data type of a
# Output: <class 'set'>
print(type(a))

Access Set

Như ta đã biết set là tập hợp không có thứ tự nên sẽ không có index Vì lý do đó không thể truy cập các phần tử bằng cách sử dụng index như các kiểu dữ liệu tập hợp khác. Nhưng có một vài tips sau đây:

  • Sử dụng loop để lặp qua các phần tử
  • Sử dụng in để kiểm tra phần tử có tồn tại hay không
# initialize a with set()
set1 = set("I love python")

# Using loop for set
# Output: lv Inetoyhp
for i in set1:
    print(i, end='')

# Using in for set
# Output: True
print('e' in set1)

Add Set

Set có thể thay đổi. Nhưng không thể sử dụng index và slicing để thêm hoặc thay đổi các phần tử. Chúng ta có thể thêm bằng các hàm built-in có sẵn:

  • add(): thêm một phần tử tại một thời điểm. Không thể thêm các phần tử có thể thay đổi như list dictionary set
  • update(): thêm nhiều phần tử tại một thời điểm. Nhận vào là list string tuple

Lưu ý: Trong tất cả trường hợp các phần tử trùng sẽ bị tránh

# initialize my_set
my_set = {1, 3}
print(my_set)

# my_set[0]
# if you uncomment the above line
# you will get an error
# TypeError: 'set' object does not support indexing

# add an element
# Output: {1, 2, 3}
my_set.add(2)
print(my_set)

# add multiple elements
# Output: {1, 2, 3, 4}
my_set.update([2, 3, 4])
print(my_set)

# add list and set
# Output: {1, 2, 3, 4, 5, 6, 8}
my_set.update([4, 5], {1, 6, 8})
print(my_set)

Delete Set

Một số hàm sau đây được sử dụng để xóa phần tử khỏi set

  • discard(): xóa phần tử trong set nhận vào đối số là giá trị
  • remove(): xóa phần tử trong set nhận vào đối số là giá trị, nếu giá trị không tồn tại trong set sẽ báo lỗi KeyError
  • pop(): xóa tùy ý một phần tử khỏi set. Gây ra lỗi KeyError nếu set rỗng
  • clear(): xóa tất cả phần tử khỏi set
# Difference between discard() and remove()

# initialize my_set
my_set = {1, 3, 4, 5, 6}
print(my_set)

# discard an element
# Output: {1, 3, 5, 6}
my_set.discard(4)
print(my_set)

# remove an element
# Output: {1, 3, 5}
my_set.remove(6)
print(my_set)

# discard an element
# not present in my_set
# Output: {1, 3, 5}
my_set.discard(2)
print(my_set)

# remove an element
# not present in my_set
# you will get an error.
# Output: KeyError

my_set.remove(2)
# initialize my_set
# Output: set of unique elements
my_set = set("HelloWorld")
print(my_set)

# pop an element
# Output: random element
print(my_set.pop())

# pop another element
my_set.pop()
print(my_set)

# clear my_set
# Output: set()
my_set.clear()
print(my_set)

Có nhiều phương thức trong set được liệt kê dưới đây:

Method Description
add() Thêm một phần tử
clear() Xóa toàn bộ phần tử
copy() Copy một set
difference() Lọc những phần khác nhau của 2 hoặc nhiều set
difference_update() Xóa tất cả phần tử của một set khác từ set này
discard() Xóa phần tử khỏi set
intersection() Giao của 2 set
intersection_update() Cập nhật set bằng việc giao của nó với set khác
isdisjoint() Trả về True nếu 2 set ko có giao
issubset() Trả về True nếu set khác không tồn tại trong set này
issuperset() Trả về True nếu set này không tồn tại trong set khác
pop() Xóa và trả về bất kỳ phần tử nào của set. Báo lỗi KeyError nếu set là empty
remove() Xóa phần tử trong set. Nếu không có tồn tại phần tử, báo lỗi KeyError
symmetric_difference() Trả về symmetric difference của 2 set
symmetric_difference_update() Cập nhật set với the symmetric difference của set đó và một set khác
union() Trả về hợp của 2 set
update() Cập nhật set với hợp của nó và set khác

Frozenset

Frozenset là một lớp mới có những đặc điểm của set Nhưng kích thước sẽ không thay đổi sau khi được gán. Tuple là list bất biến, thì Frozenset là set bất biến

Set là tập hợp có thể thay đổi nên không thể dùng làm key cho dictionary thì frozenset là bất biến nên có thể làm khóa cho dictionary

Frozenset được khởi tạo bằng hàm frozenset()

Fronzenset là bất biến nên không có thêm - cập nhât - xóa các phần tử

Frozenset có các phương thức như copy(), difference(), intersection(), isdisjoint(), issubset(), issuperset(), symmetric_difference(), union()

# Frozensets
# initialize A and B
A = frozenset([1, 2, 3, 4])
B = frozenset([3, 4, 5, 6])
>>> A.isdisjoint(B)
False
>>> A.difference(B)
frozenset({1, 2})
>>> A | B
frozenset({1, 2, 3, 4, 5, 6})
>>> A.add(3)
...
AttributeError: 'frozenset' object has no attribute 'add'
