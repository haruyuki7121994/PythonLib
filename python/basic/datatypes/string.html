Introduction

Trong Python, String là các array[...byte] đại diện cho các ký tự Unicode Có thể dùng dấu ngoặc vuông để truy cập các phần tử trong chuỗi.

Python không có kiểu dữ liệu ký tự, một ký tự đơn giản chỉ là một chuỗi có độ dài bằng 1. Máy tính không xử lý các ký tự, chúng xử lý số nhị phân. Mặc dù có thể thấy các ký tự trên màn hình. Nhưng nó được lưu trữ và thao tác dưới dạng kết hợp số 0 và 1 Quá trình chuyển đổi ký tự thành số này được gọi là mã hóa, và ngược lại gọi là giải mã. ASCII và Unicode là một số bảng mã phổ biến. Một chuỗi trong Python là một mảng byte các ký tự Unicode.

Create String

String trong Python có thể được tạo bằng cách sử dụng một dấu nháy kép, 2 dấu nháy kép hoặc 3 dấu nháy kép.

# defining strings in Python
# all of the following are equivalent
my_string = 'Hello'
print(my_string)
>>> Hello

my_string = "Hello"
print(my_string)
>>> Hello

my_string = '''Hello'''
print(my_string)
>>> Hello

# triple quotes string can extend multiple lines
my_string = """Hello, welcome to
           the world of Python"""
print(my_string)
>>> Hello, welcome to
           the world of Python

Access Characters

Chúng ta có thể truy cập các ký tự riêng lẻ bằng cách sử dụng index hoặc một loạt các ký tự bằng cách sử dụng phương pháp cắt.

Những điểm lưu ý về index:

  • Index bắt đầu bằng 0
  • Truy cập một ký tự nằm ngoài phạm vi index sẽ gây ra lỗi IndexError
  • Index là một số integer có thể dương hoặc âm. Không thể sử dụng float hoặc các loại khác, điều này sẽ dẫn đến TypeError
  • Truy cập một loạt index bằng cách sử dụng toán tử cắt :
#Accessing string characters in Python
str = 'programiz'
print('str = ', str)
>>> str =  programiz

#first character
print('str[0] = ', str[0])
>>> str[0] =  p

#last character
print('str[-1] = ', str[-1])
>>> str[-1] =  z

#slicing 2nd to 5th character
print('str[1:5] = ', str[1:5])
>>> str[1:5] =  rogr

#slicing 6th to 2nd last character
print('str[5:-2] = ', str[5:-2])
>>> str[5:-2] =  am
# index must be in range
>>> my_string[15]
...
IndexError: string index out of range

# index must be an integer
>>> my_string[1.5]
...
TypeError: string indices must be integers

Change - Delete String

String là bất biến. Có nghĩa rằng không thể thay đổi hoặc xóa một phần tử trong chuỗi ngay khi mà nó được gán giá trị. Chúng ta chỉ có thể gán giá trị khác hoặc xóa toàn bộ chuỗi.

>>> my_string = 'programiz'
>>> my_string[5] = 'a'
...
TypeError: 'str' object does not support item assignment
>>> my_string = 'Python'
>>> my_string
'Python'

Sử dụng từ khóa del để xóa toàn bộ chuỗi.

>>> del my_string[1]
...
TypeError: 'str' object doesn't support item deletion
>>> del my_string
>>> my_string
...
NameError: name 'my_string' is not defined

Escape Sequence

Nếu chúng ta muốn in một văn bản như sau He said, "What's there?" Chúng ta không thể sử dụng dấu nháy kép đơn và dấu nháy kép đôi. Điều này sẽ dẫn đến SyntaxError vì văn bản chứa cả dấu nháy đơn và dấu nháy kép.

>>> print("He said, "What's there?"")
...
SyntaxError: invalid syntax
>>> print('He said, "What's there?"')
...
SyntaxError: invalid syntax

Giải quyết vấn đề này là sử dụng nháy kép 3 hoặc sử dụng escape sequence

escape sequence bắt đầu bằng dấu \ và nội dung sau đó sẽ được hiểu theo cách khác. Ví dụ: Nếu chúng ta sử dụng dấu kép đơn để đại diện cho một chuỗi, thì một dấu nháy đơn trong chuỗi phải sử dụng ký tự \ đằng trước.

# using triple quotes
print('''He said, "What's there?"''')
>>> What's there

# escaping single quotes
print('He said, "What\'s there?"')
>>> What's there

# escaping double quotes
print("He said, \"What's there?\"")
>>> What's there

Dưới đây là danh sách escape sequence hỗ trợ trong Python.

Escape Sequence Description
\newline Backslash and newline ignored
\\ Backslash
\' Single quote
\" Double quote
\a ASCII Bell
\b ASCII Backspace
\f ASCII Formfeed
\n ASCII Linefeed
\r ASCII Carriage Return
\t ASCII Horizontal Tab
\v ASCII Vertical Tab
\ooo Character with octal value ooo
\xHH Character with hexadecimal value HH 
>>> print("C:\\Python32\\Lib")
C:\Python32\Lib

>>> print("This is printed\nin two lines")
This is printed
in two lines

>>> print("This is \x48\x45\x58 representation")
This is HEX representation

Đôi khi chúng ta muốn bỏ qua escape sequence trong một chuỗi. Để làm điều này chúng ta có thể đặt r hoặc R trước chuỗi. Điều này có nghĩa là tạo một chuỗi thô và sẽ bỏ qua tất cả escape sequence

>>> print("This is \x61 \ngood example")
This is a
good example
>>> print(r"This is \x61 \ngood example")
This is \x61 \ngood example

String like List

Bởi vì String là một array các ký tự nên có thể sử dụng chuỗi như một danh sách.

# Iterating through a string
count = 0
for letter in 'Hello World':
    if(letter == 'l'):
        count += 1
print(count,'letters found')
>>> 3 letters found
>>> 'a' in 'program'
True
>>> 'at' not in 'battle'
False
