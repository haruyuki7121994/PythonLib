Python có các hàm được tạo trước cho nhiều phép toán về toán học, logic, quan hệ, bitwise... nằm trong module operator

Hàm Mô tả Tương tự
add(a, b) Cộng 2 tham số a + b
sub(a, b) Trừ 2 tham số a - b
mul(a, b) Nhân 2 số a * b
truediv(a, b) Chia 2 số lấy thập phân a / b
floordiv(a, b) Chia 2 số lấy số nguyên nhỏ nhất a // b
pow(a, b) Lũy thừa 2 số a ** b
mod(a, b) Chia 2 số lấy phần dư a % b
lt(a, b) So sánh nhỏ hơn a < b
le(a, b) So sánh nhỏ hơn hoặc bằng a <= b
eq(a, b) So sánh bằng a == b
gt(a, b) So sánh lớn hơn a > b
ge(a, b) So sánh lớn hơn hoặc bằng a >= b
ne(a, b) So sánh không bằng a != b
and_(a, b) So sánh and a & b
or_(a, b) So sánh or a | b
xor(a, b) So sánh xor a xor b
invert(a, b) So sánh inversion ~a

Những hàm quan trọng

Hàm Mô tả Tương tự
setitem(ob, pos, val) Chỉ định giá trị ở một vị trí cụ thể ob[pos] = val
delitem(ob, pos) Xóa giá trị ở một vị trí cụ thể del ob[pos]
getitem(ob, pos) Lấy giá trị ở một vị trí cụ thể ob[pos]
setitem(ob, slice(a,b), vals) Chỉ định nhiều giá trị ở những vị trí cụ thể ob[a:b] = vals
delitem(ob, slice(a,b)) Xóa nhiều giá trị ở những vị trí cụ thể del obj[a:b]
getitem(ob, slice(a,b)) Lấy nhiều giá trị ở những vị trí cụ thể ob[a:b]
concat(ob1, ob2) Nối hai object ob1 + ob2
contains(ob1, ob2) Kiểm tra ob2 có xuất hiện trong ob1 ob2 in ob1
