Hàm any() trả về True nếu bất kỳ phần tử nào của giá trị là True. Ngược lại, trả về False

any(iterable)
Parameters:
    iterable: có thể là list, string, dictionary...
Return Value:
    True: ít nhất một phần tử là True
    False: tất cả phần tử là False hoặc empty

Sử dụng any() cho List Set Tuple

# True since 1,3 and 4 (at least one) is true
l = [1, 3, 4, 0]
print(any(l))
>>> True

# False since both are False
l = [0, False]
print(any(l))
>>> False

# True since 5 is true
l = [0, False, 5]
print(any(l))
>>> True

# False since iterable is empty
l = []
print(any(l))
>>> False

Sử dụng any() cho String

# At east one (in fact all) elements are True
s = "This is good"
print(any(s))
>>> True

# 0 is False
# '0' is True since its a string character
s = '000'
print(any(s))
>>> True

# False since empty iterable
s = ''
print(any(s))
>>> False

Sử dụng any() cho Dictionary

# 0 is False
d = {0: 'False'}
print(any(d))
>>> False

# 1 is True
d = {0: 'False', 1: 'True'}
print(any(d))
>>> True

# 0 and False are false
d = {0: 'False', False: 0}
print(any(d))
>>> False

# iterable is empty
d = {}
print(any(d))
>>> False

# 0 is False
# '0' is True
d = {'0': 'False'}
print(any(d))
>>> True

Hàm all() trả về True nếu tất cả phần tử nào của giá trị là True hoặc empty. Ngược lại, trả về False

all(iterable)
Parameters:
    iterable: có thể là list, string, dictionary...
Return Value:
    True: tất cả phần tử là True hoặc empty
    False: ít nhất một phần tử là False

Sử dụng all() cho List Set Tuple

# all values true
l = [1, 3, 4, 5]
print(all(l))
>>> True

# all values false
l = [0, False]
print(all(l))
>>> False

# one false value
l = [1, 3, 4, 0]
print(all(l))
>>> False

# one true value
l = [0, False, 5]
print(all(l))
>>> False

# empty iterable
l = []
print(all(l))
>>> True

Sử dụng all() cho String

s = "This is good"
print(all(s))
>>> True

# 0 is False
# '0' is True
s = '000'
print(all(s))
>>> True

s = ''
print(all(s))
>>> True

Sử dụng all() cho Dictionary

s = {0: 'False', 1: 'False'}
print(all(s))
>>> False

s = {1: 'True', 2: 'True'}
print(all(s))
>>> True

s = {1: 'True', False: 0}
print(all(s))
>>> False

s = {}
print(all(s))
>>> True

# 0 is False
# '0' is True
s = {'0': 'True'}
print(all(s))
>>> True
