Toán tử số học

Toán tử số học sử dụng để thực hiện các phép toán như cộng, trừ, nhân, chia...

Toán tử Ý nghĩa Ví dụ
+ Cộng hai toán hạng 15 + 4 = 19
- Trừ hai toán hạng 15 - 4 = 11
* Nhân hai toán hạng 15 * 4 = 60
/ Chia hai toán hạng, kết quả là số thực 15 / 4 = 3.75
% Chia hai toán hạng lấy phần dư 15 / 4 = 3
// Chia hai toán hạng, kết quả là số nguyên làm tròn bên trái 15 // 4 = 3
** Lũy thừa 2 toán hạng 15 ** 4 = 50625

Toán tử so sánh

Toán tử so sánh được sử dụng để so sánh các giá trị. Nó trả về True hoặc False tùy theo điều kiện.

Toán tử Ý nghĩa Ví dụ
> So sánh lớn hơn 10 > 12 is False
< So sánh nhỏ hơn 10 < 12 is True
== So sánh bằng 10 == 12 is False
!= So sánh không bằng 10 != 12 is True
>= So sánh lớn hơn hoặc bằng 10 >= 12 is False
<= So sánh nhỏ hơn hoặc bằng 10 <= 12 is True

Toán tử logic

Toán tử logic bao gồm toán tử and or not

Toán tử Ý nghĩa Ví dụ
and True nếu cả 2 toán hạng là True, False nếu 1 trong 2 toán hạng là False True and False is False
or True nếu 1 trong 2 toán hạng là True, False nếu cả 2 toán hạng là False True and False is True
not True nếu toán hạng là False, False nếu toán hạng là True not True is False

Toán tử bitwise

Toán tử bitwise hoạt động trên các toán hạng là các chữ số nhị phân. Chúng hoạt động trên từng bit.

Toán tử Ý nghĩa Ví dụ
& Bitwise AND 10 & 4 = 0 ( 0000 1010 & 0000 0100 = 0000 0000)
| Bitwise OR 10 | 4 = 14 ( 0000 1010 | 0000 0100 = 0000 1110)
~ Bitwise NOT ~10 = -11 ( ~0000 1010 = 1111 0101)
^ Bitwise XOR 10 ^ 4 = 14 ( 0000 1010 ^ 0000 0100 = 0000 1110)
>> Bitwise dịch phải 10 >> 2 = 2 ( 0000 1010 >> 0000 0010 = 0000 0010)
<< Bitwise dịch trái 10 << 2 = 40 ( 0000 1010 << 0000 0010 = 0010 1000)

Toán tử chỉ định

Toán tử chỉ định là chỉ định một giá trị cho biến

Toán tử Ví dụ Tương đương
= x = 5 x = 5
+= x += 5 x = x + 5
-= x -= 5 x = x - 5
*= x *= 5 x = x * 5
/= x /= 5 x = x / 5
%= x %= 5 x = x % 5
//= x //= 5 x = x // 5
**= x **= 5 x = x ** 5
&= x &= 5 x = x & 5
|= x |= 5 x = x | 5
^= x ^= 5 x = x ^ 5
>>= x >>= 5 x = x >> 5
<<= x <<= 5 x = x << 5

Toán tử nhận dạng

Toán tử nhận dạng bao gồm is is not Chúng được sử dụng để kiểm tra xem 2 giá trị hoặc 2 biến có nằm trên cùng một bộ nhớ hay không. 2 biến số bằng nhau không có nghĩa là chúng giống hệt nhau.

Toán tử Ý nghĩa Ví dụ
is True nếu 2 toán hạng giống hệt nhau (tham chiếu đến cùng một đối tượng) x is True
is not True nếu 2 toán hạng không giống nhau (không tham chiếu đến cùng một đối tượng) x is not True 
x1 = 5
y1 = 5
x2 = 'Hello'
y2 = 'Hello'
x3 = [1,2,3]
y3 = [1,2,3]

# Output: False
print(x1 is not y1)

# Output: True
print(x2 is y2)

# Output: False
print(x3 is y3)

Chú ý: Chúng ta thấy x1 và y1 là 2 số nguyên có cùng giá trị cũng như giống hệt nhau. x2 và y2 cũng cùng trường hợp String. Nhưng x3 và y3 là danh sách. Chúng bằng nhau nhưng không giống nhau. Bởi vì trình thông dịch định vị chúng được lưu trữ ở bộ nhớ khác nhau mặc dù chúng bằng nhau.

Toán tử thành viên

in not in là toán tử thành viên trong Python. Được sử dụng để kiểm tra liệu giá trị hoặc biến có được tìm thấy trong một chuỗi hay không (string, list, tuple, set, dictionary). Trong Dictionary chỉ có thể kiểm tra sự hiện diện của key chứ không tìm được ở value

Toán tử Ý nghĩa Ví dụ
in True nếu giá trị / biến được tìm thấy 5 in x
not in True nếu giá trị / biến không được tìm thấy 5 not in x 
x = 'Hello world'
y = {1:'a',2:'b'}

# Output: True
print('H' in x)

# Output: True
print('hello' not in x)

# Output: True
print(1 in y)

# Output: False
print('a' in y)

Chú ý: 'H' có trong x nhưng 'hello' không có trong x (Python có phân biệt chữ hoa và chữ thường). Tương tự, 1 là key và 'a' là value trong Dict y. Do đó, 'a' trong y trả về False
